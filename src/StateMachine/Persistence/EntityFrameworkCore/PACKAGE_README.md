# Welcome to the Minimal Telegram Bot project

Minimal Telegram Bot is a modern .NET framework for building Telegram Bots using simple and concise syntax inspired by ASP.NET Core Minimal APIs.

This package provides states persistence based on [Entity Framework Core](https://www.nuget.org/packages/microsoft.entityframeworkcore) for the [Minimal Telegram Bot](https://www.nuget.org/packages/MinimalTelegramBot) Framework State Machine.
